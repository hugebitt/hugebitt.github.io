Intro

안녕하세요. 이번 포스팅에서는 현재 블로그에서 포스팅 밑에 이전글, 다음글 에 대해서 카테고리별로 표시될 수 있도록 커스터마이징 하는 방식에 대해 짧게 공유드리려 합니다.

제가 사용하는 테마는 hexo-icarus-theme 라는 테마입니다. 버전은 5.1.1 을 쓰고 있습니다.
